# Quiz App – Wielojęzyczna aplikacja quizowa

Aplikacja quizowa z 6 kategoriami, 3 językami (polski, angielski, ukraiński), timerem na pytanie oraz systemem osiągnięć.

## Funkcjonalności

- ✅ Wybór kategorii: Matematyka, Przyroda, Logika, Bajki, Geografia, Sport
- ✅ 3 języki: PL, EN, UA
- ✅ Timer 10 sekund na pytanie
- ✅ Pasek postępu
- ✅ Osiągnięcia po zakończeniu quizu
- ✅ Kolorowe przyciski z efektem hover

## Wymagania systemowe

- Python 3.6 lub nowszy
- System z okiennym GUI (Windows, Linux z X11, macOS)
